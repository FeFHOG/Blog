# 我高考成绩很一般。

 说难听点，在你们现在的年级排名里，属于**老师不会多看一眼**、**同学记不住名字**的那一档。

所以你要是以为我要说什么“只要努力就能逆袭”，那你可以直接关掉。

我真正逆袭的，不是分数，是**认知**。

------

我高中的时候，就发现一件很反直觉的事

你们班里，**最拼命刷题的人**，最后未必混得最好。
 反而是那几个——

- 看起来不怎么刷题
- 成绩稳定在中上
- 但特别会“偷懒思考”的人

后来，走得最远。

当年我不理解。
 后来我明白了：**高考筛的是执行力，社会筛的是模型。**

刷题，是在别人的模型里当变量。
 认知，是自己先建模。

------

高考之后，我第一次意识到：分数只是“局部最优解”

上大学后我见过太多这样的人：

- 高考 650+，但不知道自己要干嘛
- 绩点 4.0，但不敢做任何不确定的选择
- 只会问一句话：**“这个有没有标准答案？”**

而现实是——
 **越重要的选择，越没有标准答案。**

这时候你会发现一个残酷事实：

> 高考训练的是“在封闭系统内最优解题”，
>  而人生是“在开放系统中做决策”。

这两件事，几乎是反的。

------

我真正开始“反超”的，是我不再迷信努力本身

我开始问一些，当年老师不会教你的问题：

- 为什么这个专业毕业的人，十年后差距能拉这么大？
- 为什么有些人很早就能踩中风口？
- 为什么有些人看起来不努力，但机会永远比你多？

答案很简单，但也很扎心：

**他们比你更早看懂了“游戏规则”。**

而大多数人，一辈子都在把“勤奋”当护城河。

------

给还在高中的你一句实话

如果你现在：

- 只关心排名，不关心结构
- 只问“考什么”，不问“为什么考这个”
- 只想上岸，从不想岸上是什么

那你就算高考赢了，也很可能在人生的第二回合被清算。

而真正的逆袭，往往不是某一次考试，
 而是**你什么时候开始用“系统视角”看自己的人生**。

------

我为什么敢现在这么说？

不是因为我多牛。
 而是因为我**从一个普通高考生的位置，完整走过了这条弯路**。

我清楚地知道：

- 分数解决不了方向问题
- 努力解决不了认知差
- 安全感，往往是最大的陷阱

所以我才敢告诉你：
 **你现在成绩一般，并不等于你未来没机会；
 但你现在不思考，基本等于把未来交出去了。

如果你现在还在高一 / 高二 / 高三，
 我只有一句建议：

> **别只把高考，当成人生的终点线。
>  它更像是：第一轮淘汰赛。**

真正决定你上限的，是你什么时候意识到这一点。
